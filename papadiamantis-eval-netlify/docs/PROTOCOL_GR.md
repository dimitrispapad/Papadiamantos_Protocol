# Πρωτόκολλο Αξιολόγησης (Σύνοψη)

## Στόχος
Σύγκριση 3 κατηγοριοποιήσεων (A/B/C) ως προς:
- συνοχή εντός ομάδων (coherence),
- αστοχίες (misplacements),
- διάκριση “ίδια/διαφορετική κατηγορία” μέσω ζευγών,
- συμφωνία μεταξύ ειδικών.

Οι κατηγοριοποιήσεις εμφανίζονται τυφλά ως A/B/C.

## Tasks

### 1) Cluster task (εντός ομάδας)
Για κάθε cluster:
- Κάθε έργο: Coherence 1–5 και Misplaced (Ναι/Όχι).
- Cluster: προτεινόμενη ετικέτα/θέμα + σύντομα σχόλια.

### 2) Pair task (ζεύγη)
Για κάθε ζεύγος:
- Relatedness 1–5.
- Αν Relatedness ≥ 4: καταγραφή «κοινού θεματικού στοιχείου» (free text).

## Διάταξη 9 ειδικών
- 3 primary ανά κατηγοριοποίηση.
- Shared anchors από όλες τις κατηγοριοποιήσεις (διασταύρωση).
- Crossover clusters από τις άλλες 2 κατηγοριοποιήσεις.
- Ζεύγη: ισορροπημένα positive/negative, από A/B/C.
